# 🎬 CINEAI Studio

AI Image & Video Generation Platform built with Next.js 14.

## Features
- AI Image Generation (Stable Diffusion)
- AI Video Generation (Stable Video Diffusion)
- Multi-language Support (FA, EN, AR, TR, DE)
- Stripe Payment Integration
- Cloudinary Storage
- PostgreSQL Database with Prisma

## Getting Started

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
